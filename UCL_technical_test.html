Introduction

This report presents the analysis of synthetic patient data generated using the Synthea platform. The data consists of five main datasets (patients, conditions, observations, encounters, and medications) linked by a common patient identifier, alongside three ontology dictionaries (SNOMED-CT, LOINC, and RxNorm) used to encode clinical concepts.

patients_clean <- patients %>%
  mutate(
    BIRTHDATE = as.Date(BIRTHDATE),
    DEATHDATE = as.Date(DEATHDATE)
  ) %>%
  # Exclude patients where BIRTHDATE is missing or in the future
  filter(
    !is.na(BIRTHDATE),
    BIRTHDATE <= Sys.Date()
  ) %>%
  # Exclude patients where DEATHDATE is in the future
  filter(
    is.na(DEATHDATE) | DEATHDATE <= Sys.Date()
  ) %>%
  # Exclude patients where DEATHDATE is before BIRTHDATE
  filter(
    is.na(DEATHDATE) | DEATHDATE > BIRTHDATE
  ) %>%
  # Exclude patients where patient ID appears more than once
  group_by(Id) %>%
  filter(n() == 1) %>%
  ungroup() %>%
  # Apply remaining transformations
  mutate(
    GENDER    = factor(GENDER, levels = c("M", "F")),
    RACE      = factor(RACE),
    ETHNICITY = factor(ETHNICITY),
    MARITAL   = factor(MARITAL,
                       levels = c("S", "M"),
                       labels = c("Single", "Married")),
    STATE     = factor(STATE),
    CITY      = factor(CITY),
    AGE       = as.numeric(difftime(Sys.Date(), BIRTHDATE, units = "weeks") / 52.25),
    age_group = cut(AGE,
                    breaks = c(0, 9, 19, 29,39, 49, 59, 69, 79, Inf),
                    labels = c("0-9", "10-19", "20-29", "30-39", "40-49", "50-59", "60-69", "70-79", "80+"),
                    right  = FALSE)
  ) %>%
  # Oldest living person is 122 years old, so filter out any ages above 122
filter(AGE <= 122) %>%
# Missing gender
filter(!is.na(GENDER), GENDER != "") %>%
# Invalid gender category (anything other than M or F)
filter(GENDER %in% c("M", "F"))

The raw patient dataset contained 1162 patients. The following quality control checks were applied to ensure data integrity:

  • Missing or future birthdates — patients with no recorded birthdate or a birthdate after today were excluded
  • Future or implausible death dates — patients with a death date after today were excluded as these are likely data entry errors
  • Death before birth — patients where the recorded death date preceded the birth date were excluded as this is biologically impossible
  • Duplicate patient IDs — all rows sharing a patient ID were removed to prevent double counting in analyses
  • Implausible age — patients aged over 122 years were excluded based on the verified record for the oldest person ever recorded
  • Missing or invalid gender — patients with no recorded gender or a gender value other than Male or Female were excluded

After applying these checks, 1114 patients were retained for analysis, representing an exclusion of 48 patients (4.1%). All other datasets (conditions, encounters, medications, and observations) were subsequently filtered to retain only records belonging to these 1114 patients, ensuring consistency across all analyses.

Blood pressure measurements were extracted using LOINC codes 8480-6 (systolic blood pressure) and 8462-4 (diastolic blood pressure), and BMI was extracted using LOINC code 39156-5.
